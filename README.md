# 画像分割ツール

縦長の画像を自動で 2000×900px のタイルに分割するシンプルなWebアプリです。

## 機能

- ✅ ドラッグ&ドロップで画像をアップロード
- ✅ 自動で 2000×900px に等分割
- ✅ PNG / JPEG 形式で出力
- ✅ 個別またはまとめてダウンロード可能

## 使い方

1. [ここ](https://github.com/ueda/image-splitter) を開く
2. 画像をドラッグ&ドロップ（またはクリックして選択）
3. 分割されたタイルをプレビュー
4. ダウンロード

## 技術

- HTML5 Canvas API
- Vanilla JavaScript
- ブラウザのみで処理（サーバー不要）
